# Alarmes

## O que é um Alarme?

Um alarme é uma regra que dispara uma ação quando uma condição específica é atendida.

## Componentes de um Alarme

- Condição: Parâmetro, Operador, Valor
- Ação: Notificação, Email, SMS, Webhook
- Severidade: Informacional / Aviso / Crítico

## Criando um Alarme

1. Menu → "Alarmes"
2. Clique "Novo Alarme"
3. Configure condição e ações
4. Clique "Salvar"

## Boas práticas

- Evite alarmes redundantes
- Revise thresholds periodicamente
- Teste alarmes
